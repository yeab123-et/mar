Expense tracker

Expence tracker, is a command-line Expense Tracker application written in C++. It allows a user to manage their personal finances by recording daily expenses. The program uses a struct to define an Expense including description, amount, category, and date and provides a simple menu-driven interface to add new expenses, delete existing expenses by index, and view a formatted list of all recorded expenses.Data is stored in a fixed-size array in memory and is lost upon program exit.

simple calculator

The file FullFunctionCal.java is a Java application that implements a simple graphical calculator using the Swing GUI library. The calculator is designed to perform basic arithmetic operations addition, subtraction, multiplication, and division on two user-inputted numbers. It features a custom layout including text fields for the first number, second number, and the result, along with a set of numeric buttons (0-9), operator buttons, and a clear button. The code uses an ActionListener to handle button clicks, allowing users to enter numbers by clicking buttons and compute the final result.
